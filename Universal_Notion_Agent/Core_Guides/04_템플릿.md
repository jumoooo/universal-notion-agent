# 📋 Notion 페이지 작성 템플릿 (범용)

> 💡 **범용 가이드**: 모든 MCP 지원 AI 플랫폼에서 사용 가능

## 📋 TL;DR (핵심 요약)

**목적**: AI Agent가 사용자와 소통할 때 사용할 표준 템플릿 제공

**템플릿 종류**:

1. 셋업 완료 보고
2. 진행 상황 보고
3. 에러 보고
4. 사용자 질의 (명확화 요청)
5. 최종 완료 보고

---

## 1️⃣ 셋업 완료 보고 템플릿

**사용 시점**: 초기 셋업 완료 또는 대화 시작 시

```markdown
✅ **Notion 페이지 작성 Agent 준비 완료**

**현재 상태**:

- 🔌 MCP 연결: Notion API 연결 ✅ | 브라우저 자동화 연결 ✅
- 📂 워크스페이스: `[워크스페이스 경로]`
- 🌐 브라우저: 준비 완료 ✅

**지원 기능**:

1. 마크다운 파일 → Notion 페이지 업로드
2. 부모 페이지 지정 지원
3. 대용량 파일 청크 분할 업로드
4. 자동 내용 검증

**사용 방법**:

다음과 같이 요청해주세요:

- 기본: `[파일명].md 파일을 Notion에 업로드해줘`
- 부모 지정: `[파일명].md를 [부모 페이지 URL] 하위에 업로드해줘`

업로드할 파일을 알려주세요! 📄
```

## 2️⃣ 진행 상황 보고 템플릿

### 2-1. 파일 읽기 단계

```markdown
📄 **단계 1: 파일 읽기 및 분석**

- ✅ 파일 경로 확인: `[파일 경로]`
- ✅ 파일 읽기 완료: [라인 수]줄
- ✅ 페이지 제목: `[포맷팅된 제목]`
- ✅ 청크 분할 전략: [전략 설명] ([청크 수]개 청크)

다음 단계로 진행합니다...
```

### 2-2. 페이지 생성 단계

**옵션 A (API 방식)**:

```markdown
🆕 **단계 2: Notion 페이지 생성 (API 방식)**

- ✅ 제목 생성: `[제목]`
- ✅ API로 페이지 생성 완료
- ⏳ 부모 페이지로 이동 시도 중...

[진행 중...]
```

**옵션 A → C 전환 시**:

```markdown
⚠️ **API 이동 실패 감지 → 브라우저 자동화로 전환**

**원인**: API로 부모 페이지로 이동할 수 없습니다.

**대체 방법**: 브라우저 자동화(옵션 C)로 전환하여 부모 페이지에서 직접 하위 페이지를 생성합니다.

**진행 단계**:

1. ✅ 부모 페이지 URL 구성
2. ⏳ 부모 페이지 접속 중...
```

**옵션 C (브라우저 자동화)**:

```markdown
🆕 **단계 2: Notion 페이지 생성 (브라우저 자동화)**

**부모 페이지**: `[부모 페이지 제목]`

**진행 단계**:

1. ✅ 부모 페이지 접속 완료
2. ✅ 로그인 상태 확인 완료
3. ✅ 편집 권한 확인 완료
4. ⏳ 하위 페이지 생성 중 (`/page` 명령어)...
5. ⏳ 제목 입력 중: `[제목]`

[진행 중...]
```

### 2-3. 내용 업로드 단계

```markdown
📤 **단계 4: 내용 업로드 진행 중**

**진행률**: [현재 청크]/[전체 청크] 청크 업로드 완료 ([진행률]%)

**상세**:

- ✅ 청크 1/[전체]: 업로드 완료 ([크기]자)
- ✅ 청크 2/[전체]: 업로드 완료 ([크기]자)
- ⏳ 청크 3/[전체]: 업로드 중... ([크기]자)

[잠시만 기다려주세요...]
```

### 2-4. 검증 단계

```markdown
🔍 **단계 5: 내용 검증 중**

- ⏳ 파일의 마지막 줄 확인 중...
- ⏳ 페이지 내용 추출 중...
- ⏳ 검증 진행 중...

[진행 중...]
```

## 4️⃣ 사용자 질의 템플릿 (명확화 요청)

### 4-1. 파일 경로 명확화

```markdown
❓ **파일 경로 확인 필요**

다음 파일을 찾을 수 없습니다: `[파일명]`

**워크스페이스 검색 결과**:

1. `[경로1]`
2. `[경로2]`
3. `[경로3]`

어떤 파일을 사용하시겠습니까? (번호로 답변해주세요)

또는 정확한 파일 경로를 입력해주세요.
```

### 4-2. 부모 페이지 확인

```markdown
❓ **부모 페이지 확인**

부모 페이지를 지정하지 않으셨습니다.

**옵션**:

1. 워크스페이스 루트에 생성 (부모 없음)
2. 부모 페이지 URL 또는 ID 입력

어떻게 진행하시겠습니까? (번호로 답변해주세요)
```

### 4-3. 제목 확인

```markdown
❓ **페이지 제목 확인**

**파일명**: `[파일명]`
**자동 생성된 제목**: `[포맷팅된 제목]`

이 제목을 사용하시겠습니까?

- "예" 또는 "확인": 자동 생성 제목 사용
- 다른 제목: 원하시는 제목 입력

답변해주세요.
```

## 5️⃣ 최종 완료 보고 템플릿

### 5-1. 완전 성공

```markdown
✅ **Notion 페이지 작성 완료!**

**결과**:

- 📄 페이지 제목: `[제목]`
- 🔗 페이지 URL: [페이지 URL]
- 📊 업로드된 내용:
  - 전체 라인 수: [라인 수]줄
  - 청크 수: [청크 수]개
  - 총 문자 수: [문자 수]자
- ⏱️ 소요 시간: [시간]초

**검증 결과**: ✅ 모든 내용이 정상적으로 업로드되었습니다.

페이지를 확인하시려면 위 URL을 클릭해주세요! 🎉
```

### 5-2. 부분 성공 (검증 실패 시)

```markdown
⚠️ **Notion 페이지 작성 완료 (검증 필요)**

**결과**:

- 📄 페이지 제목: `[제목]`
- 🔗 페이지 URL: [페이지 URL]
- 📊 업로드된 내용:
  - 전체 라인 수: [라인 수]줄
  - 청크 수: [청크 수]개
  - 총 문자 수: [문자 수]자

**검증 결과**: ⚠️ 최종 검증에 실패했습니다.

**권장 사항**:

1. 위 페이지 URL을 직접 확인해주세요
2. 내용이 누락된 경우 재시도하겠습니다

페이지를 확인하셨습니까? 문제가 있습니까?
```
