## 🎨 고급 프롬프트 템플릿

### 1. 여러 파일 일괄 업로드

```
[가이드 참조 방법] 를 참조하여
다음 파일들을 Notion에 업로드해줘:
1. [파일명1].md
2. [파일명2].md
3. [파일명3].md

모두 [부모 페이지 URL] 하위에 업로드해줘
```

---

### 2. 에러 발생 시 재시도

```
[가이드 참조 방법] 를 참조하여
[파일명].md 파일을 Notion에 다시 업로드해줘

이전 에러:
[에러 메시지]

이번에는 [해결 방법] 으로 진행해줘
```

---

### 3. 검증 실패 시 재업로드

```
[가이드 참조 방법] 를 참조하여
[페이지 URL] 의 내용을 확인하고
누락된 부분이 있다면 [파일명].md 의 내용을 다시 업로드해줘
```

---

### 4. 특정 부분만 업로드

```
[가이드 참조 방법] 를 참조하되
[파일명].md 파일의 [시작 라인]부터 [종료 라인]까지만
Notion에 업로드해줘
```

## 📚 상황별 프롬프트 예시

### 상황 1: 처음 사용하는 경우

```
Universal_Notion_Agent/README.md 를 먼저 읽어줘
그 다음 Core_Guides/00_메인_가이드.md 를 참조하여
노션_셋업_가이드.md 파일을 Notion에 업로드해줘
```

---

### 상황 2: 로그인이 필요한 경우

```
Universal_Notion_Agent/Core_Guides/00_메인_가이드.md 를 참조하여
먼저 Notion 로그인 상태를 확인해줘

로그인이 되어있다면
노션_작업_실행_가이드.md 파일을 Notion에 업로드해줘
```

---

### 상황 3: 에러 처리가 필요한 경우

```
Universal_Notion_Agent/Core_Guides/00_메인_가이드.md 와
Universal_Notion_Agent/Core_Guides/03_에러_처리.md 를 참조하여

[파일명].md 파일을 Notion에 업로드해줘
에러 발생 시 즉시 보고하고 해결 방법을 제시해줘
```

---

### 상황 4: 대용량 파일 업로드

```
Universal_Notion_Agent/Core_Guides/00_메인_가이드.md 를 참조하여
[파일명].md 파일을 Notion에 업로드해줘

파일 크기가 크므로:
1. 청크를 최대 300줄 단위로 분할해줘
2. 각 청크 업로드 후 1초 대기해줘
3. 진행 상황을 자세히 보고해줘
```
