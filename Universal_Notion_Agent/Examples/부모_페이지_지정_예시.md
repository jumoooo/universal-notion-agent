# 📄 부모 페이지 지정 업로드 예시

> 범용 Notion Agent를 사용하여 특정 부모 페이지 하위에 업로드하는 예시

## 🎯 시나리오: 특정 부모 페이지 하위에 업로드

### 사용자 프롬프트

```
Universal_Notion_Agent/Core_Guides/00_메인_가이드.md 를 참조하여
document.md 파일을
https://www.notion.so/[부모페이지ID]
하위에 업로드해줘
```

### 예상 진행 과정

1. **파일 읽기**:

   - 파일 경로 확인
   - 파일 내용 읽기
   - 라인 수 확인: 1,697줄
   - 페이지 제목 생성: "Document"

2. **부모 페이지 URL 파싱**:

   - 입력: `https://www.notion.so/[부모페이지ID]`
   - 추출: `[부모페이지ID]`

3. **페이지 생성 시도** (API → 브라우저 자동화):

   - API로 페이지 생성 시도
   - 부모 페이지로 이동 시도
   - ⚠️ API 이동 실패 감지
   - → **브라우저 자동화로 전환** (옵션 C)

4. **브라우저 자동화**:

   - 부모 페이지 접속
   - 로그인 상태 확인
   - 편집 권한 확인
   - `/page` 명령어로 하위 페이지 생성
   - 제목 입력: "Document"

5. **청크 분할**:

   - 전략: 1,697줄 → 300-500줄 단위
   - 청크 수: 약 4개

6. **내용 업로드**:

   - 청크 1/4 업로드 (424줄)
   - 청크 2/4 업로드 (424줄)
   - 청크 3/4 업로드 (424줄)
   - 청크 4/4 업로드 (425줄)

7. **검증**:
   - 마지막 줄 텍스트 확인
   - 페이지 내용 검증
   - 완료 보고

### 예상 결과

```
✅ Notion 페이지 작성 완료!

결과:
- 📄 페이지 제목: 노션 작업 실행 가이드
- 📁 부모 페이지: [부모 페이지 제목]
- 🔗 페이지 URL: https://www.notion.so/노션-작업-실행-가이드-xxxxx
- 📊 업로드된 내용:
  - 전체 라인 수: 1,697줄
  - 청크 수: 4개
  - 총 문자 수: ~35,000자
- ⏱️ 소요 시간: 약 30초

검증 결과: ✅ 모든 내용이 정상적으로 업로드되었습니다.
```

## 📋 지원하는 부모 페이지 URL 형식

### 1. 일반 URL

```
https://www.notion.so/[페이지ID32자리]
```

### 2. 공유 페이지 URL

```
https://[workspace].notion.site/[페이지ID32자리]
```

### 3. 앵커 포함 URL

```
https://www.notion.so/[페이지ID32자리]#[앵커ID]
```

### 4. UUID 형식 (하이픈 포함)

```
[UUID형식]
```

### 5. UUID 형식 (하이픈 없음)

```
[32자리ID]
```

**모든 형식이 자동으로 파싱됩니다!**
