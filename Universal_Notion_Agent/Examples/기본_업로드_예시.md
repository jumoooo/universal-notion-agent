# 📄 기본 업로드 예시

> 범용 Notion Agent를 사용한 기본 업로드 예시

## 🎯 시나리오: 단일 파일을 워크스페이스 루트에 업로드

### 사용자 프롬프트

```
Universal_Notion_Agent/Core_Guides/00_메인_가이드.md 를 참조하여
document.md 파일을 Notion에 업로드해줘
```

### 예상 진행 과정

1. **파일 읽기**:

   - 파일 경로 확인
   - 파일 내용 읽기
   - 라인 수 확인: 557줄
   - 페이지 제목 생성: "노션 셋업 가이드"

2. **페이지 생성** (API 방식):

   - 워크스페이스 루트에 페이지 생성
   - 제목: "Document"

3. **청크 분할**:

   - 전략: 500-2,000줄 → 200-300줄 단위
   - 청크 수: 약 3개

4. **내용 업로드**:

   - 청크 1/3 업로드 (186줄)
   - 청크 2/3 업로드 (186줄)
   - 청크 3/3 업로드 (185줄)

5. **검증**:
   - 마지막 줄 텍스트 확인
   - 페이지 내용 검증
   - 완료 보고

### 예상 결과

```
✅ Notion 페이지 작성 완료!

결과:
- 📄 페이지 제목: 노션 셋업 가이드
- 🔗 페이지 URL: https://www.notion.so/노션-셋업-가이드-xxxxx
- 📊 업로드된 내용:
  - 전체 라인 수: 557줄
  - 청크 수: 3개
  - 총 문자 수: ~12,000자
- ⏱️ 소요 시간: 약 15초

검증 결과: ✅ 모든 내용이 정상적으로 업로드되었습니다.
```
